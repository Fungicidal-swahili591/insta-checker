# Instagram Followers Checker
- This tool can track who doesnt follow you back on Instagram
- It can also track who you followed/followed you in chronological order

# How to Use
- In the .env file, enter your Instagram Session ID
- Run locally with Flask or deploy for easier access

# DISCLAIMER
- This tool is not affiliated with Instagram
- Never leak your session ID to anyone
- Use with caution, if you send too many requests your account could get banned
