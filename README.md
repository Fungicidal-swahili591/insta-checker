# Instagram Followers Checker
- This tool can track who doesnt follow you back on Instagram
- It can also track who you followed/followed you in chronological order

# How to Use
- Download this as a .zip
- Make sure requirements.txt are installed
- In the .env file, enter your Instagram Session ID
- Run it on your IDE of choice and go to http://127.0.0.1
# Alternative
- Go to Render or Replit or any other Flask deployment tools
- Upload the files into there
- Enter your session ID in the .env
- Run it and now this tool will be accessable on all devices via the link it will give you
- Do not make your app public, this will allow people to see and use your session ID
